## Common Issues & Solutions

### Issue: CSS not loading
```
Solution:
1. Check file path in HTML
2. Clear browser cache (Ctrl+Shift+Del)
3. Hard refresh (Ctrl+Shift+R)
4. Check if file exists in folder
```

### Issue: JavaScript not working
```
Solution:
1. Open Developer Tools (F12)
2. Check Console tab for errors
3. Verify script.js path
4. Check for typos
```

### Issue: Mobile looks broken
```
Solution:
1. Check viewport meta tag
2. Test on different devices
3. Check responsive CSS
4. Resize browser window
```

### Issue: Site not deploying
```
Solution:
1. Wait 2-3 minutes after push
2. Check GitHub Actions for errors
3. Verify all files are pushed
4. Check Pages settings
```
